1. Instalasi Dependensi
- Buka terminal di dalam direktori proyek `alifio-chatbot`, lalu ketik `npm install` dan enter

2. GET API, dikarenakan GitHub dan gemini melarang upload API Key secara publik
- Buka website https://aistudio.google.com/api-keys
- Klik Create API Key > Create key > Copy API Key
- Buka folder .env
- Paste key tersebut pada lines 2 setelah code ini `GEMINI_API_KEY=`

3. Buka terminal lalu ketik `node app.js` dan enter
